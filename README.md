Smart Canteen Demand Predictor 🍽️📊

An AI-powered Smart Canteen Demand Predictor that helps canteens, cafeterias, and food service providers forecast daily food demand using Machine Learning. The project analyzes historical sales, weather conditions, holidays, and customer trends to reduce food wastage and improve inventory management.

🚀 Features
📈 Food demand prediction using Machine Learning
🧠 AI-based trend analysis
🌦️ Weather-aware demand forecasting
📅 Holiday and weekend impact analysis
📊 Interactive data visualization
⚡ Real-time prediction system
💾 Dataset preprocessing and training pipeline
🖥️ User-friendly interface
🛠️ Technologies Used
Python

⚙️ Installation

Clone the repository:

git clone https://github.com/your-username/smart-canteen-demand-predictor.git

Move into the project folder:

cd smart-canteen-demand-predictor

Install dependencies:

pip install -r requirements.txt
▶️ Run the Project
Run Training
python train.py
Run Application
python app.py

or if using Streamlit:

streamlit run app.py
